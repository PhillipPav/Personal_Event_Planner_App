# Personal Event Planner App

An android studio mobile application used to convert units of measurements when travelling internationally. Created by Phillip Pavlides for SIT305 at Deakin University

## Installation
*   Open Git Bash
*   Clone the repository: `git clone https://github.com/PhillipPav/Travel_Companion_App`

## Usage
This Program allows users to create and manage a list of upcoming events. 
*   Home page displays the list of events user has created sorted automatically by date
*   The user may edit and delete exsisting events
*   The Create New event page allows users to enter data (*title, category, location, time, date*) and create an event with given data
*   It Features a bottom navigation bar to navigate between the home page and create new event page
*   The app has data persistance and will retain the events list even after closing and reopening the app
